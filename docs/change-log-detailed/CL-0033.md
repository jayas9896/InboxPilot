# CL-0033

## Summary
- Implemented OAuth code exchange for Google and Microsoft in the OAuth callback.
- Added configurable token endpoint URLs and normalized token parsing.
- Updated tests and documentation to reflect the new OAuth flow.

## Rationale
OAuth callbacks were storing authorization codes as placeholder tokens. Exchanging the code immediately yields real access/refresh tokens so future provider ingestion can use them without manual steps.
