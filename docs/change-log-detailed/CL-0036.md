# CL-0036

## Summary
- Added Outlook OAuth ingestion via API and CLI using Microsoft Graph.
- Implemented Graph message parsing helpers with tests.
- Added Microsoft Graph base URL configuration and updated UI/docs.

## Rationale
Outlook OAuth ingestion is required for parity with Gmail and for users who rely on Microsoft 365. This adds a read-only ingestion path backed by stored OAuth tokens.
